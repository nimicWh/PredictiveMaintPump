# Predictive Maint Pump (ML)
# Industrial Pump Predictive Maintenance Demo

**Predictive maintenance (PdM) for industrial pumps** using real-world sensor data. Combines industrial automation expertise with **ML and anomaly detection** to forecast failures, lead-time, and Remaining Useful Life (RUL).

---

## Features

- **Anomaly Detection**
  - **Isolation Forest (unsupervised)**: Detect early pump degradation
  - **Random Forest (supervised)**: Predict failures from labeled data

- **Predictive Metrics**
  - Lead-Time Before Failure
  - Precision / Recall / F1
  - Mean Time Between Failures (MTBF)
  - Remaining Useful Life (RUL)

- **Interactive Dashboard**
  - Step plots of anomalies vs failures
  - Lead-Time markers with **hover tooltips**
  - Critical early warnings highlighted
  - RUL chart & PLC Alarm simulation
  - Lead-Time summary table

## Installation
1. Clone the repo:

```bash
git clone https://github.com/yourusername/pump-pdm-demo.git
cd pump-pdm-demo
```

2. Install dependencies:

   Use pip install <pandas, numpy, scikit-learn, matplotlib, seaborn, streamlit, plotly>

## Usage
1. Run main.py to generate predictions:
   python pmpump_main.py

2. Launch the interactive dashboard:
   streamlit run pump_dashboard.py

3. Use sidebar controls to:
  - Adjust aggregation window
  - Zoom into sample ranges
  - Toggle ISO/RF models
  - Set critical lead-time threshold

4. Hover over markers to inspect early warnings.

## How It Works
1) Train Isolation Forest on normal pump operation
2) Train Random Forest on labeled failure data
3) Compute lead-time before failure
4) Estimate RUL per sample
5) Compute MTBF
6) Simulate PLC alarm for detected anomalies
7) Visualize interactive dashboard

## Industrial Use Cases
- Predictive Maintenance Scheduling
- Operator Dashboards for pump health
