# PredictiveMaintPump
# Industrial Pump Predictive Maintenance Demo

**Predictive maintenance (PdM) for industrial pumps** using real-world sensor data. Combines industrial automation expertise with **ML and anomaly detection** to forecast failures, lead-time, and Remaining Useful Life (RUL).

---

## 🚀 Features

- **Anomaly Detection**
  - **Isolation Forest (unsupervised)**: Detect early pump degradation
  - **Random Forest (supervised)**: Predict failures from labeled data

- **Predictive Metrics**
  - Lead-Time Before Failure
  - Precision / Recall / F1
  - Mean Time Between Failures (MTBF)
  - Remaining Useful Life (RUL)

- **Interactive Dashboard**
  - Step plots of anomalies vs failures
  - Lead-Time markers with **hover tooltips**
  - Critical early warnings highlighted
  - RUL chart & PLC Alarm simulation
  - Lead-Time summary table
